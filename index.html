⚙️ Архитектура

Мгновенный MQTT Мост

Никаких задержек. Команды из Telegram долетают до вашего компьютера менее чем за 15 миллисекунд благодаря легковесному сетевому протоколу интернета вещей. Бот работает в облаке 24/7 и не расходует батарею смартфона.

TG Bot ➔ Cloud Server ➔ Твой ПК

Быстрый старт за 3 шага

01 / СКАЧАТЬ

Запусти приложение на ПК. Программа умеет автоматически сворачиваться в системный трей Windows и поддерживает автозапуск при старте ОС.

02 / СИНХРОНИЗАЦИЯ

Открой нашего Telegram-бота, отправь команду /start и по кнопке поделись контактом для занесения в постоянную базу.

03 / АВТОРИЗАЦИЯ

Введи номер в настройках приложения на ПК, нажми "Запросить код" и укажи полученные в чате бота цифры подтверждения.
